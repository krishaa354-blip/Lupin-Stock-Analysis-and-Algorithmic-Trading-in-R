# Lupin Stock Analysis and Algorithmic Trading in R

## Project Overview

This project focuses on analyzing **10 years of Lupin stock market data** using **R Programming**.  
The project demonstrates financial data analysis, data visualization, time series forecasting, and algorithmic trading strategies using technical indicators and statistical models.

The project is designed for academic and learning purposes in the fields of:

- Financial Analytics
- Time Series Analysis
- Stock Market Prediction
- Algorithmic Trading
- Data Visualization

---

# Objectives

- Analyze historical stock market data of Lupin
- Visualize stock price movements and trends
- Apply technical indicators like Moving Averages
- Perform Time Series Analysis
- Build ARIMA forecasting models
- Generate Buy/Sell trading signals
- Understand algorithmic trading concepts

# Dataset

The dataset contains approximately **10 years of historical stock data** for Lupin including:

- Date
- Open Price
- High Price
- Low Price
- Close Price
- Trading Volume

# Project Workflow

```text
Load Data
   ↓
Data Cleaning
   ↓
Data Visualization
   ↓
Moving Average Analysis
   ↓
Time Series Analysis
   ↓
ARIMA Forecasting
   ↓
Algorithmic Trading
   ↓
Buy/Sell Signals
```


# How to Run the Project

1. Open RStudio
2. Load the R script
3. Run the script step by step
4. Select the Lupin Excel dataset when prompted
5. View generated charts and forecasts
